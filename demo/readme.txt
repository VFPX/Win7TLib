List of Folders:
----------------
1) "Resources" folders contains all shared resources between the demo applicaitons

2) "VCX" folder contains the full demo application using the win7tlib_visual.vcx visual helper class

3) "NoVCX" folder contains the full demo application using the program defined helper class.

4) "Simple" folder contains a super simple demo application with easy to follow source code.

5) "Grouping" folder contains simple demo applications for controlling grouping behavior.
